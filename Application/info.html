Welcome in the world of Nono Le Robot
Who's Nono ?

Nono le robot est robot qui a pour objectif d'explorer le monde. Il doit survivre, en trouvant de la nourriture, et explorer le monde.

How to use Nono

Nono est un simulateur interactif. Il repose sur l'aide de l'utilisateur pour lui modifier son comportement. Lancer l'application, et modifier les valeurs dans le graphique Ã  droite !

Comportement inverse - se dirige vers les arbres, et esquive les pommes : [ 10,0,0,10,0,10,10,0 ]
